Guide du GitHub

Vous trouverez dans ce répertoire le projet en NLP, où nous avons travaillé en équipe sur une solution
pour pouvoir lire des Emails. Le but est de connaître la nature du mail, s'il est un SPAM ou un non-SPAM.

L'étude de la base de données est expliquée dans le notebook_EDA.

Le modèle est crée et expliqué dans notebook_model.

Le notebook_predict est une synthèse de l'ensemble du projet avec un "Step by Step" qui permet de tester le modèle.

Vous pouvez voir notre planning ainsi que la présentation du projet !

Enfin notre modèle, notre corpus de mots et notre méthode vectorizer sont conservés dans un format pickle.

Notre application se trouve dans le "app.py" et est accompagné des éléments nécessaires pour pouvoir déployer sur Streamlit.
